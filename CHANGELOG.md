# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-05-11

### Added
- Initial release of NexGen PDF Viewer
- Professional PDF viewing component for React
- Annotation support:
  - Highlight text
  - Underline
  - Note/comment annotations
  - Strikeout
  - Box drawing
  - Circle drawing
  - Freehand drawing
- Dark mode and high-contrast support
- Advanced text search
- Multiple view modes (single, double, continuous)
- Zoom controls (fit-page, fit-width, custom)
- PDF rotation support
- Sidebar with thumbnail navigation
- Toolbar with annotation tools
- SQLite backend for annotation storage
- Real-time collaboration with 5-second polling
- Undo/Redo support for annotations
- History tracking
- TypeScript support with full type definitions
- Responsive design for mobile, tablet, and desktop

### Features
- 🚀 Fast Rendering with PDF.js
- 🎨 Professional annotation tools
- 🌓 Dark mode support
- 📱 Fully responsive
- 🔍 Advanced search functionality
- 📂 Local file upload and remote URL support
- ⚙️ Customizable theming
- 📊 Pagination and thumbnails
- 📝 Comprehensive TypeScript types
